- 🎨 **Diseño moderno y responsivo** - Compatible con desktop, tablet y móvil

## 🚀 Instalación y Uso

### Prerrequisitos
- Node.js (v16 o superior)
- npm o yarn

### Instalación

1. **Clona el repositorio**
   ```bash
   git clone https://github.com/mariorocha/portfolio.git
   cd portfolio
   
2. **instala las dependencias**
    npm install
    yarn install
3. **Inicia el servidor de desarrollo**
   npm run dev
    # o
    yarn dev
4. **Abre tu navegador**
   http://localhost:5173

    npm run dev      # Inicia el servidor de desarrollo
    npm run build    # Construye la aplicación para producción
    npm run preview  # Previsualiza la build de producción
    npm run lint     # Ejecuta ESLint para revisar el código

# 🌐 Deploy

## Netlify (Recomendado)
1. Conecta tu repositorio de GitHub
2. Configura el build command: `npm run build`
3. Configura el publish directory: `dist`
4. Deploy automático en cada push

## Vercel
1. Instala Vercel CLI: `npm i -g vercel`
2. Ejecuta: `vercel`
3. Sigue las instrucciones

## GitHub Pages
1. Instala gh-pages: `npm install --save-dev gh-pages`
2. Agrega script en package.json: `"deploy": "gh-pages -d dist"`
3. Ejecuta: `npm run build && npm run deploy`
